## 프로젝트 개요

코로나19 이후 급증한 비대면 교육 수요에 대응하기 위해 개발한 직업훈련기관 전용 학습관리시스템(LMS)입니다. 실시간 화상 강의, 학습 진도 관리, 평가 시스템을 통합한 올인원 솔루션입니다.

## 주요 기능

### 1. 실시간 화상 강의
- WebRTC 기반 화상회의
- 화면 공유 및 녹화
- 실시간 채팅 및 Q&A
- 출석 자동 체크

### 2. 학습 관리
- 커리큘럼 관리
- 학습 진도 추적
- 과제 제출 시스템
- 성적 관리

### 3. 평가 및 인증
- 온라인 시험 시스템
- 부정행위 방지 기능
- 수료증 자동 발급
- NCS 기준 평가

### 4. 관리자 대시보드
- 실시간 학습 현황
- 통계 및 리포트
- 훈련생 관리
- 강사 관리

## 기술 스택

**프론트엔드:**
- Vue.js 3
- Vuetify
- WebRTC
- Socket.io

**백엔드:**
- Node.js
- Express
- MongoDB
- Redis

**인프라:**
- AWS EC2
- S3 Storage
- CloudFront CDN
- Docker

## 성과

- **도입 기관**: 15개 직업훈련기관
- **사용자 수**: 5,000명 이상
- **만족도**: 4.3/5.0
- **학습 완료율**: 85% 달성

## 주요 고객사

- 한국직업능력개발센터
- 서울직업전문학교
- 경기직업훈련원
- 외 12개 기관

## 특징

1. **HRD-Net 연동**: 고용노동부 HRD-Net 시스템과 자동 연동
2. **모바일 지원**: 반응형 디자인으로 모든 디바이스 지원
3. **보안 강화**: SSL 인증서, 개인정보 암호화
4. **맞춤형 커스터마이징**: 기관별 요구사항 반영
