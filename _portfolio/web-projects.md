## 프로젝트 개요

2003년부터 2009년까지 드림아이티 기술연구소 대표로서 수행한 대학 웹사이트 구축 프로젝트들입니다. 총 10개 이상의 대학 웹사이트를 성공적으로 구축했습니다.

## 주요 프로젝트

### 1. 세종대학교 통합 포털 시스템
- **기간**: 2008년 3월 - 2008년 12월
- **규모**: 5억원
- **주요 기능**:
  - 학사 정보 시스템
  - 전자 도서관 연동
  - 모바일 캠퍼스
  - 통합 검색 시스템

### 2. 진주교육대학교 홈페이지 리뉴얼
- **기간**: 2007년 6월 - 2007년 11월
- **규모**: 2억원
- **주요 기능**:
  - 반응형 웹 디자인
  - CMS 시스템 구축
  - 다국어 지원
  - 접근성 인증 획득

### 3. 중앙대학교 입학처 웹사이트
- **기간**: 2009년 1월 - 2009년 6월
- **규모**: 3억원
- **주요 기능**:
  - 온라인 원서 접수
  - 입시 정보 시스템
  - 실시간 상담 채팅
  - 모의 지원 시스템

### 4. 강남대학교 e-Learning 플랫폼
- **기간**: 2006년 9월 - 2007년 3월
- **규모**: 2.5억원
- **주요 기능**:
  - 동영상 강의 시스템
  - 과제 제출 관리
  - 온라인 시험
  - 학습 분석 도구

## 기술 스택

### 초기 프로젝트 (2003-2006)
- PHP 5.x
- MySQL 5.0
- Apache 2.2
- JavaScript/jQuery

### 후기 프로젝트 (2007-2009)
- Java/Spring Framework
- Oracle 10g
- Tomcat/WebLogic
- Ajax/Web 2.0

## 프로젝트 특징

### 1. 웹 접근성
- 한국형 웹 접근성 지침(KWCAG) 준수
- 장애인 차별 금지법 대응
- 크로스 브라우징 지원

### 2. 성능 최적화
- 로드 밸런싱 구현
- CDN 활용
- 데이터베이스 최적화
- 캐싱 전략 수립

### 3. 보안
- SSL 인증서 적용
- SQL Injection 방지
- XSS 공격 대응
- 개인정보 암호화

## 성과

- **총 프로젝트 수**: 15개
- **누적 계약액**: 30억원 이상
- **평균 만족도**: 4.5/5.0
- **유지보수 계약률**: 80%

## 수상 및 인증

- 2008년 웹 어워드 코리아 교육 부문 우수상 (세종대학교)
- 2007년 웹 접근성 우수 사이트 인증 (진주교육대학교)
- 2009년 굿 디자인 어워드 (중앙대학교)
